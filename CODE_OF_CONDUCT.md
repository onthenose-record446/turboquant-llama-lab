# Code of Conduct

## Our Pledge

We want this project to be useful, technically honest, and welcoming to people who are
building, benchmarking, or studying long-context inference systems.

Participants are expected to:

- be respectful and constructive
- focus criticism on code, data, and claims
- avoid harassment, personal attacks, and hostile behavior
- communicate uncertainty honestly, especially around benchmark claims

## Expected Behavior

- share reproducible evidence when making technical claims
- label experimental results clearly
- assume good intent while still holding a high bar for rigor
- help keep discussions actionable and useful for contributors

## Unacceptable Behavior

- harassment, insults, or intimidation
- bad-faith benchmarking or misleading performance claims
- publishing private information without consent
- spam or deliberately disruptive behavior

## Enforcement

Project maintainers may remove, edit, or reject contributions and discussions that do
not follow this code of conduct.

## Scope

This code of conduct applies to project spaces, issue threads, pull requests, and
other public community interactions related to this repository.
